How to use employee manager:

Welcome!

The website allows you to view and add emplyee details, including their ID, name, role, and salary. Follow these instructions to get started:

Accessing employees:

To view the list of employees, please press the 'See all employees' button to be redirected

Add an employee:

To add an employee, please input the employee's details in the corresponding text boxes, then press 'Add employee'

Press the 'How to use this site' button to access this page for further reference

Logging out:

To log out, simply press the 'Log out' button, you will then be redirected to the log in page

Something looks off? Simply press Command + Shift + R to fix!
